# 📖 Overview

This project is a **Library Management System** developed in **C++** using the concepts of **Object Oriented Programming (OOP)**.

The system helps users manage different library items like:

* 📘 Books
* 🎬 DVDs
* 📰 Magazines

This project demonstrates:

* ✅ Abstraction
* ✅ Encapsulation
* ✅ Inheritance
* ✅ Polymorphism
* ✅ Exception Handling
* ✅ Dynamic Memory Allocation

---

# 🚀 Features

✨ Add Book
✨ Add DVD
✨ Add Magazine
✨ Display All Items
✨ Check Out Item
✨ Return Item
✨ Exception Handling
✨ Dynamic Memory Management
✨ Menu Driven Interface

---

# 🧠 OOP Concepts Used

| Concept            | Usage                                                |
| ------------------ | ---------------------------------------------------- |
| Class & Object     | Used for library item management                     |
| Abstraction        | Abstract base class `LibraryItem`                    |
| Inheritance        | `Book`, `DVD`, `Magazine` inherit from `LibraryItem` |
| Polymorphism       | Virtual functions & pointer array                    |
| Encapsulation      | Private data members with getters/setters            |
| Exception Handling | Invalid inputs handled using try-catch               |

Clone the repository
